## [v0.1.0] - 2025-01-15 ([`ecac47e`](https://github.com/grokify/presentation-spec/commit/ecac47e))

### Highlights

- Go-first specification library for data-driven presentations with JSON Schema generated from Go types
- HTML and PDF rendering with 12 widget types for dashboards and executive presentations
- Live development server with hot reload for rapid iteration
